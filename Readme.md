# 設定方法

①まずアセットをインポートします

②シーンにモデルを配置して、各SkinnedMeshRendererを複製します

③複製したSkinnedMeshRendererの各マテリアルをSSSMaskにします

④元のマテリアルのシェーダーをSSSにします

⑤いいかんじにします
